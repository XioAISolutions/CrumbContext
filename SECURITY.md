# Security policy

CrumbContext handles prompts, logs, identifiers, and potentially sensitive project context. Do not include private production data in public issues.

For security-sensitive reports, contact XIO AI Solutions privately before public disclosure. Include a minimal reproduction, affected version, and impact.

Current alpha limitations:

- generated bundles are written to the chosen local output directory;
- exact-anchor sidecars intentionally contain extracted exact values;
- users are responsible for securing those output files;
- provider network adapters are not included in v0.1.
