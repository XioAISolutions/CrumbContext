# Launch checklist

The complete launch copy, hashtags, repository topics, social preview, and posting sequence live in [`LAUNCH_KIT.md`](LAUNCH_KIT.md).

## Release gates

- [x] standalone package layout
- [x] tests pass
- [x] exact anchors preserved in the bundled benchmark
- [x] authority and recent turns remain exact
- [x] image and text-only policies self-verify
- [x] wheel and source distribution build
- [x] `twine check` passes
- [x] README labels estimates honestly
- [x] social preview and repository artwork prepared
- [x] contribution, conduct, security, citation, and issue templates prepared
- [ ] GitHub repository created
- [ ] CI and CodeQL pass on GitHub
- [ ] social preview uploaded and topics applied
- [ ] PyPI trusted publisher configured
- [ ] `v0.1.0` GitHub release published

Do not publish provider-billed savings claims until a same-request counterfactual harness exists.
