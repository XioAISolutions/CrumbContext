# Brand guide

## Core line

> Give every AI the context it needs—not the entire conversation.

## Supporting line

> Exact facts stay exact. Old context takes the cheapest explainable lane.

## Personality

CrumbContext should feel technically serious without looking like another enterprise dashboard. The crumb mascot carries the playful identity; the writing stays precise, testable, and honest.

## Visual system

- Background: near-black navy
- Violet: authority and routing intelligence
- Cyan: movement, observability, and output
- Warm crumb gold: the mascot and human-friendly identity
- Green: verified checks only
- Orange: image lane or lossy transformation warnings

## Assets

- `logo.svg`: square repository/avatar mark
- `hero.svg`: README hero
- `social-preview.svg`: GitHub and social-link preview
- `benchmark-proof.svg`: reproducible bundled benchmark result

Do not place fabricated testimonials, download counts, savings claims, or partner logos into project assets.
